# Shared Terminology

- Brain: the top-level durable operating context for a large project, org, or initiative.
- Workspace: the scoped area where agents coordinate around concrete work.
- Task: an actionable unit of work with explicit ownership and state.
- Agent Session: the active runtime identity for an agent participating in a workspace.
- Workspace Message: append-only communication for humans and agents.
- Context Entry: durable structured memory related to tasks, decisions, artifacts, or observations.
