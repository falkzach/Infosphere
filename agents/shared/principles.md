# Shared Principles

- Use Infosphere as the shared operational memory and coordination system.
- Keep durable collaborative state in tasks, workspace messages, and context entries rather than ad hoc local notes.
- Prefer explicit contracts over hidden conventions.
- Keep changes small, verifiable, and reversible.
- When behavior is unclear, surface the ambiguity rather than guessing silently.
- Close loops: if work is started, drive it through validation and clear outcomes.
